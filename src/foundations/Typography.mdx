## Notes

**Display size is container-query driven** — `clamp(36px, 13cqw, 104px)`. The `cqw`
unit needs an ancestor with `container-type: inline-size`; the specimen above sets
it on itself, so the headline scales with the docs panel rather than the viewport.

**Fredoka is the headline face**, not Bricolage Grotesque. The font families here
declare system fallbacks — Fredoka and JetBrains Mono need to be loaded by the
consuming app (or added as a webfont in `.storybook/preview-head.html`) to render
exactly as designed.
