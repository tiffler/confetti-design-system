# Color

Every swatch below is read from `build/portfolio/tokens.json` for the skin and mode
selected in the toolbar — nothing on this page is hardcoded. Switch **Mode** in the
toolbar to see the semantic layer repoint while the token names stay put.

Swatches run **lightest to darkest** by relative luminance. Translucent tokens are
flattened onto the page surface first, since that is what renders.

**Only combinations that clear 4.5:1 appear here** — the AA threshold for body text.
Anything below it is left off the page entirely rather than shown as a failure, so
every ratio you see is one you can use. Specimens render on `surface-page` rather than
the Storybook docs canvas, so the text colors are measured against the surface they
are actually meant for.
